# Imot2.bg - Система за управление на недвижими имоти

**Imot2.bg** е уеб базирано приложение, разработено с FastAPI, в което брокери могат да публикуват обяви за имоти, а клиенти - да търсят имоти и да запазват часове за огледи.

## Основни функционалности

* **Автентикация и роли:** Регистрация и вход за Клиенти, Агенти и Админи.
* **Управление на имоти:** Създаване, редактиране на обяви.
* **Качване на изображения:** Възможност за добавяне на снимки към всеки имот.
* **Система за резервации:** Клиентите могат да заявяват огледи за специфични дати.
* **Чат в реално време:** Комуникация между клиенти и брокери.
* **Административен панел:** Верификация на агенти, статистика и модерация на ревюта.
* **Търсене и филтриране:** Динамично търсене по локация, цена и тип имот.

## Използвани технологии

* **Backend:** FastAPI
* **Database:** SQLAlchemy
* **Templates:** Jinja2 & Tailwind CSS
* **Testing:** Pytest & Coverage
* **Authentication:** Cookie-based auth

### 2. Създаване на виртуална среда
* python -m venv .venv
* source .venv/bin/activate  # За Linux/macOS
* .venv\Scripts\activate     # За Windows

* pip install -r requirements.txt

### 4. Стартиране
* uvicorn main:app --reload
* Приложението ще бъде достъпно на http://127.0.0.1:8000

* pytest --cov=. --cov-report=html
